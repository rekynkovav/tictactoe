# 🎮 Крестики-нолики с Telegram ботом

Игра "Крестики-нолики" с интеграцией Telegram бота для отправки уведомлений о победах и промокодов.

## ✨ Особенности

- 🎮 Игра против компьютера с AI логикой
- 🤖 Telegram бот для уведомлений
- 🎁 Промокоды при победе
- 📊 Статистика игр
- 🐳 Docker контейнеризация
- 🗄️ PostgreSQL база данных
